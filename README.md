# Smart Attendance System
A Java-based Smart Attendance System developed using Java Swing and SQLite.

## Features
- Teacher login
- Generate unique session codes
- Student attendance using session code
- Manual attendance entry
- Attendance stored in SQLite database
- Simple Java Swing GUI

## Technologies Used
- Java
- Java Swing
- SQLite
- JDBC

## How to Run
1. Download the project.
2. Add the SQLite JDBC library.
3. Compile and run:

```
javac SmartAttendanceSystem.java
java SmartAttendanceSystem
```
